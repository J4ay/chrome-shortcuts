# Chrome Shortcuts

Chrome Shortcuts is a Chrome extension that opens custom websites with keyboard shortcuts.

## Features

- Three configurable shortcuts
- Edit shortcut URLs in the extension popup
- Keyboard shortcuts can be reassigned in and deactivated [extension shortcuts settings](chrome://extensions/shortcuts)
- Option to open shortcuts in a new tab or a new window

## Setup

1. Open `chrome://extensions` in Chrome.
2. Turn on Developer mode.
3. Click Load unpacked.
4. Select this project folder.

## Usage

1. Click the extension icon.
2. Enter up to three website URLs.
3. Click Save Shortcuts.
4. Press one of the keyboard shortcuts to open the saved URL.

## Default shortcuts

- `Ctrl+Shift+1` opens Shortcut 1
- `Ctrl+Shift+2` opens Shortcut 2
- `Ctrl+Shift+3` opens Shortcut 3

On macOS, the defaults are `Command+Shift+1`, `Command+Shift+2`, and `Command+Shift+3`.
